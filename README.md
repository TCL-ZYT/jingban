# 境班 - 全真职场模拟器

> 一款沉浸式职场模拟游戏，在架空世界「恒洲联邦」中开启你的职场人生

## 📋 产品简介

**境班**是一款单文件HTML职场模拟游戏，玩家将以新人身份加入恒洲科技，体验从实习生开始的完整职场生涯。游戏融合了任务执行、人际社交、绩效考评、职级晋升、跳槽转行等真实职场要素，通过塔罗牌人设系统为每个角色赋予独特的性格特质，让每一次职场旅程都与众不同。

## 🌍 世界观

本游戏为完全架空虚构作品，所有世界观、人物、机构、事件均设定于**恒洲联邦**，不涉及任何现实国家、政府机构、企业及个人。请以轻松心态体验职场人生。

## ✨ 核心特色

### 🎴 塔罗人设系统
- 4张命运塔罗牌（命运、因果、机缘、选择）随机抽取
- 获得 **3个正向增益标签** + **1个负面DEBUFF标签**
- 标签永久绑定角色，贯穿全职场生涯
- 影响任务完成率、迟到概率、绩效评分、剧情触发

| 商务销售 | 销售、市场、商务岗位 |

# English:
# Jingban - Full Immersion Workplace Simulator

> An immersive workplace simulation game where you begin your career in the fictional world of Hengzhou Federation

## 📋 Product Overview

**Jingban** is a single-file HTML workplace simulation game. Players take on the role of a new hire at Hengzhou Technology, experiencing a complete career journey starting from intern. The game combines real-world workplace elements such as task execution, interpersonal networking, performance reviews, rank promotions, and career switching. Through the Tarot Persona System, each character is endowed with unique personality traits, making every career journey distinct.

## 🌍 World Setting

This game is a work of complete fiction. All worldbuilding, characters, organizations, and events are set in the **Hengzhou Federation** and do not involve any real countries, government agencies, corporations, or individuals. Please enjoy your workplace journey with a lighthearted attitude.

## ✨ Core Features

### 🎴 Tarot Persona System
- Draw randomly from 4 fate tarot cards (Destiny, Karma, Opportunity, Choice)
- Receive **3 positive buff tags** + **1 negative debuff tag**
- Tags are permanently bound to your character throughout your entire career
- Affects task completion rate, tardiness probability, performance scores, and story triggers

### 🏢 6 Industries to Choose From

| Industry | Suitable Roles |
|----------|----------------|
| Internet Operations | Operations, New Media, Product positions |
| Administration | Reception, Admin, HR, Back-office positions |
| Business & Sales | Sales, Marketing, Business Development |
| Finance & Cashier | Finance, Accounting, Audit positions |
| Engineering / Mining Infrastructure | Engineering, Technical, Project Management |
| Manufacturing Back-office | Production, Supply Chain, Quality Assurance |

### 📈 Complete Growth System
- **Rank System**: P1 Intern → P8 Executive, 8 promotion tiers total
- **Performance System**: Four-dimensional scoring — Work Efficiency, Task Completion, Team Collaboration, Work Quality
- **Salary System**: Dynamically adjusts with rank promotions, performance affects year-end bonus
- **Certification System**: Earn rank certificates to unlock higher privileges

### 👥 Authentic Workplace Relationships
- 8 colleagues with unique personalities and individual trait tags
- **Intimacy System**: Stranger → Nodding Acquaintance → Regular Colleague → Friendly Partner → Close Friend → Trusted Comrade
- Chat interactions, collaborative tasks, gossip sessions — build your professional network
- Collaborative tasks unlock at intimacy ≥ 60

### 📝 Rich Task System
- Pick up tasks from the task library at your pace — 3 difficulty levels: Beginner / Standard / Professional
- Task extension request system for better work rhythm management
- Random debuff events add workplace unpredictability
- Complete team tasks with colleagues for dual rewards + intimacy boost

### 💼 Career Development System
- **Job Hopping System**: Switch between 6 industries, 3 company sizes (Small Private Enterprise / Mid-sized Company / Large Corporation)
- **Resignation System**: Complete resignation workflow experience
- **Resume System**: Tracks every chapter of your career journey

## 🎮 How to Play

### New Player Guide
1. Sign the compliance statement to enter the game
2. 5-step character creation: Basic Info → Choose Industry → Choose Department → Education Background → Tarot Draw
3. Onboarding flow: Meet colleagues → Review to-do list → Start working

### Daily Gameplay
- **Daily Check-in**: Attendance affects performance score
- **Complete Tasks**: Earn EXP and performance points
- **Interact with Colleagues**: Chat to increase intimacy
- **Collaborate**: Complete projects together with teammates
- **Submit Requests**: Leave, overtime, reimbursement and more

### Promotion Requirements
- Sufficient tenure
- Required performance rating
- Accumulated project experience
- Corresponding rank certification earned

## 🎨 Interface Design

- Realistic phone frame design for immersive experience
- Deep blue theme with particle animation on splash screen
- 5 bottom navigation tabs: Messages, Workbench, Approvals, Network, Profile
- Elegant card-based layout with iconized feature entry points
- Smooth page transitions and modal animations

## 🔧 Technical Features

- **Single File**: One HTML file with embedded CSS + JavaScript, play instantly
- **Responsive Design**: Optimized for phone frame display ratio
- **Tech Stack**: Tailwind CSS + Iconify icon library
- **No Installation Required**: Play directly in any browser

## 🎯 Target Audience

- Students who want to experience workplace life in advance
- New professionals curious about different industries and roles
- Casual gamers who enjoy simulation and management games
- Professionals who want to test scenarios in a virtual world
